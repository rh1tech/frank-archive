# FRANK Archive

Official page: **[frank.rh1.tech](https://frank.rh1.tech/)** — hub for all FRANK boards and firmware.

Archive of FRANK board revisions — schematics, layouts, and reference assets for each released and prototyped variant of the FRANK family.

Each subdirectory corresponds to a specific board revision, including the FRANK, MicroFRANK, MiniFRANK, TinyFRANK, TurboFRANK, and related accessory boards (HDMI-to-VGA, USB-to-PS/2).
